\textbf{Chi\_Square}\par
A comprehensive Python chi-square calculator covering the majority of statistical computation problems encountered in genetics courses, including goodness-of-fit tests, contingency table tests (with optional Yates' correction), and Hardy-Weinberg equilibrium tests.
